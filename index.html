System Design
Six-Module Architecture
Each module handles one problem and passes its output to the next. The fusion engine sits above all three sensing layers simultaneously rather than treating cybersecurity as an afterthought.
MODULE 01
Visual Intelligence
YOLOv8 aerial object detection producing structured counts of vehicles, persons, and boats from UAV imagery.
MODULE 02
EKF State Estimation
Extended Kalman Filter tracking UAV position. The innovation residual — how surprised the filter is by each GPS reading — is the primary anomaly signal.
MODULE 03
Cyber Anomaly Detection
Random Forest classifier trained on real GPS receiver data. Identifies three distinct spoofing attack types. Hardened with adversarial training.
MODULE 04
Dempster-Shafer Fusion
Combines vision, cyber, and context evidence into one explainable threat score using formal evidential reasoning. Context-aware: the same detection means different things in different zones.
MODULE 05
Report Generator
Converts structured fusion output into auditable plain-English intelligence reports. Every sentence traces back to a specific input field.
MODULE 06
Operator Dashboard
Streamlit + Plotly ops-center interface with threat gauge, EKF residual timeline, evidence breakdown charts, and live report generation.
Experiment
Adversarial Robustness
Standard supervised classifiers for GPS spoofing detection are vulnerable to adversarial inputs, small perturbations that cause misclassification without being detectable in raw telemetry. This experiment quantifies that vulnerability and demonstrates its mitigation through adversarial training.
Model ROC-AUC False Positive Rate Clean Recall Recall Under Attack Verdict
Standard Random Forest 0.9874 0.0605 0.8653 0.2957 FRAGILE
Gradient Boosting 0.9923 0.0100 — — UNTESTED
Adversarially Trained RF 0.9875 0.0531 0.8334 1.0000 ROBUST
Finding
Adversarial training restored recall under attack from 0.2957 to 1.0000 with negligible impact on ROC-AUC (0.9874 to 0.9875) and a reduction in false positive rate from 6.1% to 5.3%. This is consistent with adversarial machine learning findings in UAV intrusion detection literature and demonstrates that the vulnerability is practically mitigable without sacrificing detection performance.
Data
Dataset
Trained and evaluated on the Aissou 2022 GPS spoofing dataset (Mendeley Data, DOI: 10.17632/z7dj3yyzt8.3), containing 510,530 samples of real GPS receiver measurements across four classes: authentic signals and three distinct spoofing attack types. The dataset contains 13 GPS signal features including carrier-to-noise ratio, Doppler shift, carrier phase, and pseudorange measurements.
